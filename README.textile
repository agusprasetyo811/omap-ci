
h1. omap-ci

Template Library in Codeigniter like Omap Templating

h2. Setup

Place all this file into Codeigniter Source.
* Setup @development.properties@ to set Codeigniter destination  
* Run @ant remote.compile@ to build all

h2. Structure

* <b>Appliction</b>
** @config/config.php@
** @controllers/dasbor.php@
** @libraries/omap.php@
** @views/omap-ci/pages/my_pages.php@
** @views/omap-ci/modules/my_modules.php@

* <b>System/</b>
** @core/config.php@

* <b>Template/</b>
** @omap-ci/index.php@
** @omap-ci/images@
** @omap-ci/style/style.css@
** @omap-ci/js/@

h2.  Usage

h3. Setup Theme

* Open @application/config/config.php@ 
** Setup @$config['theme']@ any name. Default is <b>omap-ci</b>

h3. Omap-ci Usage

* Installing omap-ci libraries you can see @controllers/dasbor.php@
** @$this->load->library('omap')@

* Omap-ci setup title to set title of websites
** @$this->omap->title('Welcome Omap-Ci')@

* Omap-ci setup type to set type of class (pages or module)
** @$this->omap->type('pages')@
** @$this->omap->type('modules')@

* Omap-ci setup label to set label. This is usage to put the class type pages or module to the template.
** @$this->omap->label('login')@ 
** At @template/omap-ci/index.php@ it will initialized @{LOGIN}@  

* Omap-ci setup display to set display.
** @$this->omap->display('my_files',$any_data)@ 
** my_files is name of file that saved in modules or pages dependent the type of class 

* Path for images
** style @../images/file_images.jpg@
** index @images/file_images.jpg@

* Path for Js and Style 
** put @{STYLE}@ it same as @http://BASE_URL/template/THEME_NAME/style/@
** put @{JS}@ it same as @http://BASE_URL/template/THEME_NAME/js/@

* Title for website
** put @{TITLE}@ at @<title>...</title>@
